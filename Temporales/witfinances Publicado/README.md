# 💰 Calculadora de Salario Neto + Gestor de Finanzas

Esta es una aplicación web todo-en-uno que combina una **calculadora de salario neto quincenal** (Florida 2025) con un **gestor de ingresos, gastos y tarjetas de crédito**.

👉 **[🚀 Ver demo en línea](https://alberthoma.github.io/calculadora-finanzas/)**

---

## 📸 Captura

![Demo](https://alberthoma.github.io/calculadora-finanzas/demo.png)

---

## 🚀 Funcionalidades principales

- ✅ Registro de ingresos, gastos y tarjetas
- ✅ Clasificación de gastos por categoría
- ✅ Calculadora de salario neto con deducciones:
  - Impuestos federales
  - FICA (Social Security + Medicare)
  - 401(k)
  - Préstamos estudiantiles
  - Seguro médico
- ✅ Todo integrado en una sola interfaz amigable
